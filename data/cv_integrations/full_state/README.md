# CV-интеграция full_state

Заготовка для датасетов, меток, предсказаний и результатов экспериментов, где
CV-target соответствует полному state-вектору в стиле LunarLander.

Целевые колонки:

- `x`
- `y`
- `vx`
- `vy`
- `theta`
- `angular_velocity`
- `left_leg_contact`
- `right_leg_contact`

Текущий файл `data/labels.csv` содержит только метки позы, поэтому перед
обучением этого варианта нужно будет сгенерировать или оценить метки скорости и
контактов.
