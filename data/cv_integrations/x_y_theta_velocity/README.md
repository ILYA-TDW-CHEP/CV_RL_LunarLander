# CV-интеграция x_y_theta_velocity

Заготовка для датасетов, меток, предсказаний и результатов экспериментов, где
CV-target включает позу аппарата и скорость.

Целевые колонки:

- `x`
- `y`
- `theta`
- `vx`
- `vy`
- `angular_velocity`

Текущий файл `data/labels.csv` содержит только метки позы, поэтому перед
обучением этого варианта нужно будет сгенерировать или оценить метки скорости.
