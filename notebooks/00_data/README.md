# Ноутбуки для данных

Данная папка содержит ноутбуки, которые генерируют, проверяют, очищают или
разбивают датасеты кадров LunarLander.

`01_generate_images.ipynb` содержит workflow генерации датасета. Путь к
директории данных по умолчанию: `../../data`, где изображения пишутся в
`images/`, а метки — в `labels.csv`.

Сами `.npy` изображения и `labels.csv` не коммитятся в git. Если репозиторий
только что склонирован, сначала запустите этот notebook, чтобы восстановить
локальный датасет.

Запускайте notebook из его собственной папки, чтобы относительные пути
разрешались корректно.
