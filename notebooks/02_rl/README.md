# RL-ноутбуки

Используйте эту папку для reinforcement-learning экспериментов, которые работают
с наблюдениями из симулятора или с CV-derived наблюдениями.

Идеи для будущих ноутбуков:

- диагностика обучения DQN
- оценка политики
- сравнение режимов наблюдений
- анализ replay buffer'ов и checkpoint'ов
