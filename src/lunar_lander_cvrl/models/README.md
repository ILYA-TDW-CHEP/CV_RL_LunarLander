# Версии моделей

Здесь будут храниться будущие реализации CV- и RL-моделей.

На данный момент для CV части доступны:

- `ResNet18` в двух режимах (только координаты, а также координаты + угол)
- `SimpleCNN` так же в двух режимах

Для RL части доступны:

- `DQN` из библиотеки `stable-baselines3`. Возможно обучение во всех режимах
